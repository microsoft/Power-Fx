#SETUP: PowerFxV1CompatibilityRules

// With Power Fx V1 rules, if follows the "first-type rule" - the result of the expression
// will be the type of the first argument

>> If(1>0, {a:1}, {b:2})
{a:1}

>> If(1<0, {a:1}, {b:2})
{a:Blank()}

>> If(false, {x:1, y:1}, {x:2, z:2})
{x:2,y:Blank()}

>> If(true, {x:1, y:1}, {x:2, z:2})
{x:1,y:1}
